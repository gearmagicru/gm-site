## Установка

Предпочтительный способ установки этого расширения — через [composer](http://getcomposer.org/download/), 

или выполнить:

```
php composer.phar require --prefer-dist gearmagicru/gm-site "*"
```

или добавить

```
"gearmagicru/gm-site": "*"
```

в раздел "require" вашего файла `composer.json`.
